Requisitos del hardware

  • LilyGo T5 4.7" S3
  • ESP32-S3
  • Cable USB-C

Como instalar

  1. Conecta el dispositivo via cable USB-C a tu computadora
  2. Haz clic en "Instalar Firmware" arriba
  3. Selecciona el puerto COM (ej: COM3, /dev/ttyUSB0)
  4. Espera a que termine la instalacion (~1 min)
  5. Configura conectandote a WiFi "WeatherStation-Setup"

Solucion de problemas

  • "No port selected" = No seleccionaste puerto. Reconecta el cable y reintenta.
  • No aparece puerto = Instala drivers CH340/CP2102 o usa otro cable USB.
  • Falla la conexion = Entra en modo bootloader: manten BOOT, presiona RST, suelta RST, suelta BOOT.
